# TP JPA Hibernate Spring Data

## Description
Mini projet Spring Boot démontrant l'utilisation de JPA/Hibernate
avec Spring Data pour la gestion d'une base de données.

## Technologies utilisées
- Java 17
- Spring Boot 3.x
- Spring Data JPA
- Hibernate
- H2 Database (en mémoire)
- Lombok
- Maven
